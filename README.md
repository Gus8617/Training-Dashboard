# 🚀 Training Dashboard & Adaptive Coach

Un tableau de bord complet pour le suivi de l'entraînement, synchronisant les activités **Strava** et les données de santé **Garmin**. Inclut un bot **Telegram** pour les rapports quotidiens et les résumés d'activités en temps réel.

## 🛠️ Tech Stack
- **Frontend**: React + Vite
- **Backend**: Node.js + Express
- **Base de données**: SQLite (`better-sqlite3`)
- **Automation**: `node-cron`
- **Gestionnaire de process**: PM2
